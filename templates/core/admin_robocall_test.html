{# ═══ Справка ═══ #}
Как это работает
  1. В личном кабинете zvonok.com зайдите в Настройки профиля → API и сгенерируйте Public key.
  2. Создайте (или выберите существующую) кампанию с роботом: Автодозвон → Новая кампания → загрузить аудиоролик или задать текст. Скопируйте её ID.
  3. Введите оба значения в форму настроек выше и сохраните.
  4. В форме «Позвонить на номер» укажите телефон и нажмите «Позвонить» — робот перезвонит клиенту по сценарию этой кампании.
  5. Результат звонка zvonok вернёт через webhook (если настроен в кампании) или можно посмотреть в панели zvonok.com.
Стоимость: каждый тестовый звонок тарифицируется с вашего баланса zvonok.com.
